# Incident Response Runbook (Draft)
## Severity levels
SEV1: platform signing key compromise / receipt forgery risk / global outage  
SEV2: partial outage, export/verify degradation  
SEV3: minor bug, limited tenant impact

## Mandatory steps
1. Declare SEV, open incident doc
2. Freeze changes
3. Communicate status cadence (30 min for SEV1)
4. Contain, eradicate, recover
5. Postmortem in 72 hours with corrective actions
